Markdown Language

Overview

Markdown is a lightweight markup language with plain text formatting syntax. Its design allows it to be converted to many output formats, but the original tool by the same name only supports HTML.[8] Markdown is often used to format readme files, for writing messages in online discussion forums, and to create rich text using a plain text editor.

Headings in Markdown

To create heading in markdown laguage you should add # symbols in front of the heading depending on its level.

Examples:

# This is heading 1 in Markdown
### This is heading 2 in Markdown
...
###### This is heading 6 in Markdown

After conversion to HTML:

This is heading 1 in Markdow

This is heading 2 in Markdown

...

This is heading 6 in Markdown

Lists

Unordered Lists

In order to create an undordered list in Markdown you should add a * or - symbols in front of your list items.

* first item
* second item
* third item
        
- first item
- second item
- third item
  • first item
  • second item
  • third item

Ordered list

In order to create an undordered list in Markdown you should add numbering to your list items.

1. first item
2. second item
3. third item
  1. first item
  2. second item
  3. third item

Definition Lists

To create a definition list you should use the following syntax 

term 1
: definition 1
term 1
definition 1

Tables

In order ot create a tabel you have to use the | character.

Example:

| First cell|Second cell|Third cell|
|-----------|-----------|----------|
|First      |  Second   | Third    |
|First      |  Second   | Third    |

Will result in the following table:

First cell Third cell
First Second Third
